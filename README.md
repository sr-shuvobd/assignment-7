# 🤝 Friend Tracker App

A simple React-based web application to track your interactions with friends.  
You can log calls, texts, and video check-ins, and visualize them using charts.

---

## 🚀 Features

- 📋 View all friends in card layout  
- 🔍 See detailed friend information  
- 📞 Add interactions (Call, Text, Video)  
- 📜 Timeline to track all activities  
- 📊 Analytics chart (Pie/Donut) for interactions  
- ⚠️ Prevent duplicate interactions  
- 📱 Fully responsive design  

---

## 🛠️ Technologies Used

- React.js  
- React Router  
- Tailwind CSS + DaisyUI  
- Recharts (for charts)  
- React Toastify  

---

## 📊 Pages

- Home Page (Friends list)  
- Details Page (Friend info + actions)  
- Timeline Page (Interaction history)  
- Stats Page (Analytics chart)  
- 404 Error Page
